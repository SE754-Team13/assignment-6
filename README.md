# CodeCraft – OOP Learning Platform

## Prerequisites

- Node.js 18+ and npm
- Java 17+

Maven does not need to be installed separately because the backend includes the Maven wrapper.

## Run the project

From the repository root:

```bash
npm install
npm run dev
```

## Proposed Feature

Enhance each lesson with bite-sized steps that break concepts into short, focused chunks, one idea per screen. Each step pairs a brief explanation with a Java code example. Users advance at their own pace via Previous/Next buttons, with a progress bar tracking position. A completion screen confirms when the lesson is finished.
