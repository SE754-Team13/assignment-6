# Performance Test Artifacts

This directory stores all performance testing files required for assignment submission.

## Folders

- `plans/` - JMeter `.jmx` test plans.
- `results/` - performance outputs such as `.csv` result files.
- `screenshots/` - screenshots from JMeter views such as Summary Report or View Results Tree.
- `data/` - request payloads, CSV datasets, or other input files used by JMeter.

## Naming conventions

- Use clear, descriptive file names.
- Prefer lowercase names with hyphens, for example `login-load-test.jmx`.
- Keep related result and screenshot names aligned with their plan names where possible.

## Submission reminder

Commit all performance test artifacts in this directory so they are included with the assignment submission.
